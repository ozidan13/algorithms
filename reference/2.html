ARCHITECTURE

Class Structure

Invoker

Triggers the command (e.g., Button, Remote).

Command

Interface. Declares execute().

Concrete Command

Implements execute(). Calls Receiver logic.

Receiver

Does the actual work (e.g., Light, AC).
