Algorithms Learning Repository

Interactive visualizations and implementations of various algorithms based on the "Grokking Algorithms" book. Organized as a 4-week learning roadmap to help you understand fundamental algorithms and data structures through visual, interactive examples.

Explore Algorithms

Week 1: Introduction to Algorithms

The first week covers fundamental concepts in algorithms, providing a solid foundation for more advanced topics.

Topics Covered

  • Introduction to Algorithms
  • Binary Search
  • Big O Notation
  • Selection Sort
  • Recursion
Binary Search

Interactive visualization of the binary search algorithm on sorted arrays.

Time Complexity: O(log n)
Space Complexity: O(1)

Features

  • Step-by-step visualization
  • Interactive search
  • Code explanation

Selection Sort

Visual representation of the selection sort algorithm for array sorting.

Features

  • Code walkthrough

Big O Visualization

Week 2: Arrays, Linked Lists, and Recursion

Topics Covered

  • Arrays and Their Limitations
  • Linked Lists (Singly and Doubly Linked)
  • Advanced Recursion Techniques

  • Memory Management in Data Structures
Array Operations

Interactive visualization of array operations and their performance characteristics.

Linked List

Week 3: Quicksort, Hash Tables, and Graphs

This week explores more advanced algorithms and data structures that form the foundation of many complex applications.

Topics Covered

  • Quicksort Algorithm and Analysis
  • Hash Tables and Hash Functions
  • Collision Resolution Strategies
  • Graph Representation
  • Breadth-First Search (BFS)
  • Depth-First Search (DFS)
Quicksort Visualization

Interactive visualization of the quicksort algorithm and its partitioning process.

Time Complexity: O(n log n) average case, O(n²) worst case
Space Complexity: O(log n)

Features

  • Animated pivot selection
  • Partitioning visualization
  • Performance comparison
Explore
Hash Table

Implementation and visualization of hash tables and their operation.

Time Complexity: O(1) average for insert/search/delete, O(n) worst case
Space Complexity: O(n)

Features

  • Interactive hash function demonstration
  • Collision resolution visualization
  • Performance comparison
Explore
Graph Representation

Visual comparison of different ways to represent graphs in data structures.

Features

  • Interactive creation of graphs
  • Conversion between adjacency lists and matrices
  • Comparison of space and time requirements
Explore
Graph Traversal

Visualization of graph traversal algorithms (BFS and DFS).

Time Complexity: O(V + E) for both BFS and DFS
Space Complexity: O(V)
